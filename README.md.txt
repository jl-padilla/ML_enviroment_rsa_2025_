# 📊 Proyecto EDA - Exploratory Data Analysis
ANALISIS MEDIOAMBIENTAL rsa2025


## Descripción
Este proyecto tiene como objeto analizar los formulario de las 1573 entidades que han conseguido sello RESPONSABILIDAD SOCIAL ARAGON 2025 y determinar la agrupación de estas entidades más proclive a contratar servicios de asesoría medioambiental. 
Los formularios están disponibles en la web: IAF https://www.aragonempresa.com/empresas-sello-rsa/
Posteriormente sobre los datos obtenidos de clasificación proponemos un modelo para seleccionar otras entidades fuera de este ámbito.

📊 EDA Project - Exploratory Data Analysis
ENVIRONMENTAL ANALYSIS rsa2025

Description
The objective of this project is to analyze the forms of the 1,573 entities that have obtained the RESPONSABILIDAD SOCIAL ARAGON 2025 (RSA 2025) certification and determine the group of these entities most likely to hire environmental consulting services.
The forms are available on the IAF website: IAF https://www.aragonempresa.com/empresas-sello-rsa/
Subsequently, based on the classification data obtained, we propose a model to select other entities outside this scope.
